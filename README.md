# qvim
## Installation
### Install Neovim
<https://github.com/neovim/neovim>
### Install dependencies
- ripgrep
    - <https://github.com/BurntSushi/ripgrep>
- fd (only macOS)
    ```zsh
    brew install fd
    ```
### Pull
```shell
git clone https://github.com/oneqit/qvim.git ~/.config/nvim
```
