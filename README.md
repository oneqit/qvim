# qvim
## Installation
### Install Neovim
<https://github.com/neovim/neovim>
### Pull
```shell
git clone https://github.com/oneqit/qvim.git ~/.config/nvim
```
## Shortcuts
### vim
```
gg First
G Last
Ctrl+U PgUp
Ctrl+B PgUp
Ctrl+D PgDn
Ctrl+F PgDn
0 Home
^ Home (non-whiespace)
$ End
v Visual Mode
V Visual Line Mode
```
### neovim + plugins
```
Ctrl+E :Neotree toggle
Ctrl+<Arrow> pane navigation
za fold
zM fold all
zR unfold all
```
